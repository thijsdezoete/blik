Frequently Asked Questions

What is 360-degree feedback?

360-degree feedback is a performance review method where employees receive anonymous feedback from multiple sources including peers, managers, direct reports, and self-assessment. This multi-perspective approach provides comprehensive insights into performance and behavior.

How does Blik ensure reviewer anonymity?

Blik uses cryptographically secure UUID-based tokens. Reviewer identities are never stored in the database. The system cannot link responses to specific individuals, ensuring true anonymity. Even administrators cannot trace feedback to reviewers.

What are the system requirements for self-hosting?

Blik requires Docker and Docker Compose. Minimum recommended: 2GB RAM, 2 CPU cores, 10GB storage. Runs on Linux, macOS, or Windows with Docker Desktop. PostgreSQL 15 database included in Docker setup.

Can I customize the questionnaires?

Yes. Blik includes default 360 templates and supports creating custom questionnaires. Question types include Likert scales, multiple choice, and free text. Organize questions into sections and mark as required or optional.

Is Blik GDPR compliant?

Self-hosted Blik gives you full control over data residency and processing, enabling GDPR compliance. Host in EU regions, implement data retention policies, and exercise right to deletion. Our EU SaaS offering is hosted in GDPR-compliant data centers.

How is Blik different from commercial 360 tools?

Blik is open source and self-hosted. No per-user fees, no data lock-in, no third-party access to sensitive feedback. You own your data completely. Customize the codebase to your exact requirements without vendor restrictions.

What email providers are supported?

Blik supports any SMTP email service including Gmail, Office 365, SendGrid, Amazon SES, Mailgun, and custom SMTP servers. Configure during initial setup wizard or update later in settings.

Can reviewers save progress and return later?

Yes. Reviewers can partially complete forms and return using the same token link. Progress is automatically saved. Once submitted, tokens are locked to prevent duplicate responses.

What happens if there are too few responses?

Blik enforces minimum response thresholds (default: 3 per category) to protect anonymity. If thresholds are not met, category results show "Insufficient responses" instead of data that could identify individuals. Self-assessments always display separately.

How are reports generated and shared?

Administrators manually trigger report generation after closing a review cycle. Reports aggregate responses by category with charts and visualizations. Reviewees access reports via secure token links sent by email.

Can I migrate from another 360 tool to Blik?

Yes. Blik uses standard PostgreSQL database. Import historical data via Django management commands or database tools. Enterprise customers receive migration assistance. Contact us for migration planning support.

Is there a mobile app?

Blik is mobile-responsive and works in any browser. No separate app installation required. Reviewers and reviewees access everything via mobile web browsers with optimized layouts.

What license is Blik released under?

MIT License. Use commercially, modify, distribute, and sublicense freely. No copyleft restrictions. Ideal for businesses wanting to customize or white-label the solution.

How do I get support for self-hosted installations?

Community support via GitHub issues and discussions. Documentation includes deployment guides and troubleshooting. Enterprise customers receive priority email and video call support with SLA guarantees.

Can I white-label Blik for clients?

Yes. MIT license permits white-labeling. Customize branding, domain, email templates, and UI. Consulting firms can deploy isolated instances per client with custom branding for each.

What databases are supported?

PostgreSQL 15+ is the primary supported database. Docker Compose setup includes PostgreSQL automatically. Other Django-compatible databases may work but are not officially tested or supported.

How often is Blik updated?

Regular updates with bug fixes, security patches, and feature additions. Release notes published on GitHub. Self-hosted users control update timing. SaaS customers receive automatic updates with zero downtime.

Can I run multiple review cycles simultaneously?

Yes. Create unlimited concurrent review cycles for different reviewees. Each cycle is independent with its own questionnaire, reviewers, and timeline. No restrictions on active cycles.

What analytics and insights does Blik provide?

Reports include aggregated scores by competency area, category breakdowns (peer/manager/direct report), self vs others comparisons, distribution charts, strength and development area identification, and anonymous text comment grouping.

Is there an API for integrations?

Not in the current version. Blik focuses on web interface usability for MVP. API access is on the roadmap for future releases. Contribute API development via GitHub pull requests if needed sooner.
