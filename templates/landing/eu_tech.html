Technical Questions Answered

Which EU hosting providers are recommended?

Germany: Hetzner Cloud (excellent price/performance, 100% renewable energy, Nuremberg datacenter)

France: OVH Cloud (strong privacy reputation, Strasbourg/Roubaix datacenters)

Netherlands: TransIP (Dutch privacy laws)

Finland: UpCloud (Nordic data protection standards)

Any provider with EU datacenters works. Minimum specs: 2 vCPU, 4GB RAM, 40GB SSD.

What about email sending for invitations?

Use EU-based SMTP providers: Mailgun EU (Dublin datacenter), SendGrid EU, Amazon SES EU (Frankfurt), or your own mail server. Configure SMTP in the setup wizard. All email processing stays in EU. Templates are simple text, no tracking pixels.

Can we integrate with our existing LDAP/SSO?

Currently, admin authentication uses Django's built-in system. LDAP/SSO integration is on the roadmap. For now, admins log in with username/password. Reviewers never authenticate (token-based access only). You can modify the authentication backend since it's open source.

How do we demonstrate GDPR compliance to auditors?

Blik includes documentation showing:

1) Data flow diagrams proving EU residency
2) Technical measures (encryption, access controls, anonymity)
3) Organizational measures (admin audit logs, retention policies)
4) Data subject rights implementation (export, deletion)

Open source code allows auditors to verify technical controls directly. For GDPR compliance guidance, see GDPR.eu and European Data Protection Board.

What's the migration path from Lattice/Culture Amp?

Export your employee list from existing tool as CSV. Import into Blik via Django admin or API. Historical review data can be archived separately (Blik focuses on new cycles, not historical reporting). Plan 2-4 weeks for parallel running during transition period. Most teams switch completely after first successful cycle.

How do backups work with GDPR retention requirements?

PostgreSQL dumps to EU-based storage (Hetzner Storage Box, S3 EU buckets). Set automated retention policies matching your data retention requirements. Implement deletion scripts for GDPR "right to be forgotten" requests. All backup data stays in EU jurisdiction. Encrypted backups recommended for extra security.

Can we run this completely air-gapped behind our firewall?

Yes. Disable external AI API calls and use local models. Configure SMTP to your internal mail server. All web traffic stays internal. No outbound internet access required. Perfect for highly regulated environments or companies with strict security policies.
